Introduction

The Command Prompt (cmd) is a command-line interpreter available on Windows operating systems. It allows users to interact with the system by executing commands, scripts, and batch files, making it a powerful tool for managing files, troubleshooting, and automating tasks. Originally developed as a part of MS-DOS, cmd has evolved into a robust utility for modern Windows systems. It is widely used by developers, IT professionals, and power users for tasks ranging from file management to network troubleshooting.

When you first open Command Prompt (cmd) on Windows, you'll typically see a screen that looks like this:

C:\Users\YourUsername>
Why Use Command Prompt?

Command Prompt is favored for its:

  1. Efficiency: Perform tasks faster than with the graphical interface.
  2. Automation: Automate repetitive tasks using batch scripts.
  3. Power: Access advanced tools and features not available through the GUI.
  4. Flexibility: Execute commands for system management, file processing, and networking.

The echo command in cmd is used to display a line of text or output. By typing the words directly after echo, they will be displayed in the terminal. example:

echo Efficiency Automation Power Flexibility

the Output:

Efficiency Automation Power Flexibility
File and Directory Management

Commands for managing files and folders:

  • assoc: Display or change file type associations.
  • cd or chdir: Change directory.
  • cls: Clear the screen.
  • copy: Copy one or more files.
  • del or erase: Delete files.
  • dir: Display directory contents.
  • exit: Exit the Command Prompt.
  • fc: Compare two files or sets of files.
  • mkdir or md: Create a new directory.
  • move: Move files or directories.
  • rmdir or rd: Remove a directory.
  • rename or ren: Rename a file or directory.
  • tree: Graphically display folder structure.
  • type: Display the contents of a text file.
  • xcopy: Copy files and directories, including subdirectories.
System Information and Management

Commands to check and manage the system:

  • chkdsk: Check a disk and display a status report.
  • diskpart: Open the disk partition utility.
  • driverquery: List all installed drivers.
  • echo: Display messages or enable/disable command echoing.
  • hostname: Display the computer's hostname.
  • ipconfig: Display network configuration.
  • mode: Configure system devices.
  • path: Display or set the system path.
  • sfc: Scan and repair system files.
  • systeminfo: Display detailed system information.
  • tasklist: Show currently running processes.
  • taskkill: Kill a process by its PID or name.
  • ver: Display the Windows version.
  • vol: Display the volume label and serial number.
Disk and Drive Management

Commands for managing drives and file systems:

  • diskpart: Manage disk partitions.
  • format: Format a disk for use with Windows.
  • label: Change the volume label.
  • mountvol: Create, delete, or list a volume mount point.
Network and Internet Utilities

Commands for networking and connectivity:

  • arp: Display or modify the ARP table.
  • ftp: Open an FTP session.
  • getmac: Display MAC addresses.
  • ipconfig: Display IP configuration.
  • netstat: Show network statistics and active connections.
  • nslookup: Query DNS servers.
  • ping: Test network connectivity.
  • route: Display or modify routing tables.
  • tracert: Trace the route to a host.
File Searching and Text Processing

Commands for finding files and processing text:

  • find: Search for a text string in a file.
  • findstr: Search for strings using regular expressions.
  • for: Loop through a set of files.
  • more: Display text one screen at a time.
  • sort: Sort text output.
Batch and Scripting Utilities

Commands useful for writing scripts and automating tasks:

  • call: Call a batch file from another batch file.
  • choice: Provide a selection menu in batch scripts.
  • for: Loop through items in a batch script.
  • goto: Jump to a labeled line in a script.
  • if: Perform conditional operations.
  • pause: Pause script execution and display a message.
  • set: View, set, or remove environment variables.
  • start: Open a new command prompt window to run a program.
Security and Permissions

Commands for managing user permissions and system security:

  • cacls: Display or modify file access control lists (ACLs).
  • cipher: Manage encryption and decryption of files.
  • net user: Manage user accounts.
  • runas: Run a program as another user.
  • whoami: Display the current logged-in user.
Utility and Troubleshooting

Commands for troubleshooting or additional utilities:

  • attrib: Display or modify file attributes.
  • chkntfs: Display or modify disk check settings.
  • fc: Compare two files.
  • fsutil: Manage file systems and drives.
  • help: Display help for commands.
  • shutdown: Shut down or restart the computer.
How to View All Commands in cmd?

To get a complete list of commands directly in Command Prompt, type:

help

For detailed information about a specific command, use:

[command] /?
